#  Credit Card Fraud Detection using Machine Learning

##  Overview
This project detects fraudulent credit card transactions using a classification model. It focuses on handling imbalanced datasets and improving model performance.

##  Tech Stack
- Python
- Pandas
- Scikit-learn

##  Workflow
- Data loading and preprocessing
- Handling imbalanced dataset using resampling
- Feature scaling
- Model training using Logistic Regression
- Model evaluation (Confusion Matrix, Precision, Recall)

##  Key Highlights
- Successfully handled imbalanced data problem
- Improved classification performance
- Evaluated using multiple metrics

##  Future Improvements
- Use real-world dataset (Kaggle)
- Try advanced models (Random Forest, XGBoost)
- Apply SMOTE for better balancing
